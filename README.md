# 🌙 Picrew Dark Mode & Landscape

A custom UserStyle that adds a universal dark mode to [Picrew](https://picrew.me) and completely overhauls the image maker interface for desktop users by forcing a full-screen, landscape layout. 

![preview of picrew dark mode & landscape view](https://file.garden/Z7zzresaFi0SdpFv/picrew.me_en_image_maker_2795963%20(1).png)
Picrew in example is by @Maddel_Art

## ✨ Features

* **Universal Dark Mode**: Applies a smart invert filter across the entire website to reduce eye strain. Images, canvases, SVGs, and color swatches are protected from the inversion so the art retains its true colors.
* **Desktop Landscape View**: Tired of scrolling up and down on a wide monitor? This script triggers on maker pages (`/image_maker/`) for screens larger than 768px, splitting the screen into a two-column layout:
    * **Left Side**: A maximized, responsive canvas that retains its 1:1 aspect ratio.
    * **Right Side**: A scrollable menu for parts and color selections.
* **Ad & Clutter Removal**: Hides side advertisements, footers, and unnecessary mobile UI elements to give you a clean workspace.
* **Optimized UI**: Fixes thumbnail grids and ensures color palette swatches have borders so dark colors don't vanish into the background.

## 🛠️ Installation

To use this theme, you will need a user style manager extension installed in your browser. 

1. Install **Stylus** for your browser:
   * [Stylus for Chromium](https://chrome.google.com/webstore/detail/stylus/clngdbkpkpeebahjckkjfobafhncgmne)
   * [Stylus for Firefox](https://addons.mozilla.org/en-US/firefox/addon/styl-us/)
2. Open the Stylus extension and click **Write style** for `picrew.me`.
3. Copy the CSS code from `style.user.css` in this repository and paste it into the Stylus text editor.
4. Click **Save** on the left side.

## 🖥️ Compatibility

* **Browsers**: Chromium & Firefox browsers.
* **Devices**: Works best in landscape-mode for tablets & desktops. Mobile not tested but might work.
